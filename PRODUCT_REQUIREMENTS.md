# 灵枢智能排盘 - 产品需求文档

## 📱 应用定位
**灵枢智能排盘** - 专业级国潮易学排盘工具
- 六爻排盘（文王卦）
- 奇门遁甲排盘
- 本地智能解析（无需联网）
- 国潮美学设计

## 🎯 核心功能

### 1. 六爻排盘模块
#### 1.1 起卦方式
- [ ] **铜钱摇卦**（3D 动画效果）
  - 模拟真实铜钱抛掷
  - 六次摇卦生成六爻
  - 记录每次的正反面
- [ ] **数字起卦**
  - 输入任意数字
  - 按规则转换为卦象
- [ ] **时间起卦**
  - 使用当前时间
  - 农历转换干支

#### 1.2 卦象展示
- [ ] **本卦**（主卦）
  - 六爻卦画
  - 卦名（如"乾为天"）
  - 卦辞原文
- [ ] **变卦**（之卦）
  - 动爻标记
  - 变卦卦象
- [ ] **互卦**（中间过程）
- [ ] **错卦**、**综卦**

#### 1.3 详细解析
- [ ] **卦辞解析**
  - 原文 + 白话翻译
  - 吉凶判断
- [ ] **爻辞解析**
  - 每爻详细解释
  - 动爻重点说明
- [ ] **五行分析**
  - 世应定位
  - 六亲配置（父母、兄弟、妻财、子孙、官鬼）
  - 五行生克关系
- [ ] **用神判断**
  - 根据所问事项选择用神
  - 旺衰分析
- [ ] **断卦建议**
  - 综合判断吉凶
  - 行动建议

### 2. 奇门遁甲排盘模块
#### 2.1 基础排盘
- [ ] **时间输入**
  - 公历/农历切换
  - 真太阳时校正
  - 节气计算（定阴阳遁）
- [ ] **方位选择**
  - 八方位选择
  - 度数精确到度

#### 2.2 盘面展示
- [ ] **九宫格布局**
  - 地盘（固定）
  - 天盘（值符）
  - 人盘（八门）
  - 神盘（八神）
- [ ] **二十四山**
- [ ] **六十甲子**

#### 2.3 要素解析
- [ ] **八门解析**
  - 开门、休门、生门、伤门、杜门、景门、死门、惊门
  - 门的吉凶
  - 门的旺相休囚
- [ ] **九星解析**
  - 天蓬、天任、天冲、天辅、天英、天芮、天柱、天心、天禽
  - 星的吉凶
- [ ] **八神解析**
  - 值符、螣蛇、太阴、六合、白虎、玄武、九地、九天
- [ ] **格局判断**
  - 吉凶格（龙回首、鸟跌穴等）
  - 特殊格局识别

#### 2.4 应用建议
- [ ] **择吉建议**
  - 适宜事项
  - 禁忌事项
- [ ] **方位建议**
  - 有利方位
  - 不利方位
- [ ] **时间建议**
  - 吉时推荐
  - 凶时规避

### 3. 公共功能
#### 3.1 历史记录
- [ ] 保存排盘记录
- [ ] 记录分类（六爻/奇门）
- [ ] 快速回溯

#### 3.2 收藏功能
- [ ] 收藏经典卦例
- [ ] 添加个人笔记

#### 3.3 分享功能
- [ ] 生成精美图片
- [ ] 分享卦象链接
- [ ] 导出 PDF 报告

## 🔧 技术实现

### 5.1 数据结构
```typescript
// 六爻卦象
interface LiuYaoGua {
  benGua: Gua;      // 本卦
  bianGua?: Gua;    // 变卦
  huGua?: Gua;      // 互卦
  dongYao: number[]; // 动爻位置
  shiYing: {        // 世应
    shi: number;
    ying: number;
  };
}

// 奇门盘面
interface QiMenPan {
  juShu: number;    // 局数
  yinYang: 'yin' | 'yang';
  baMen: string[];  // 八门
  jiuXing: string[]; // 九星
  baShen: string[];  // 八神
  geJu: string[];   // 格局
}
```

### 5.2 核心算法
1. **六爻起卦算法**
   - 铜钱概率模拟
   - 卦象生成
   - 动爻判断

2. **奇门排盘算法**
   - 定阴阳遁
   - 定局数
   - 找值符值使
   - 天盘转动
   - 八门转动

### 5.3 性能要求
- 首次启动 < 3 秒
- 排盘计算 < 1 秒
- 离线可用
- 数据本地存储
