## ⚠️ 已知问题

### TypeScript 配置警告
**现象**: 编译时出现 JSX 相关警告  
**影响**: 不影响运行，仅影响 IDE 提示  
**原因**: tsconfig.json 配置不完整  
**解决方案**: 
```json
{
  "compilerOptions": {
    "jsx": "react-native",
    "esModuleInterop": true
  }
}
```

### 类型定义冲突
**现象**: `@types/node` 和 React Native 类型冲突  
**影响**: 不影响运行  
**解决方案**: 升级相关类型定义包
