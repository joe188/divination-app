## 🔧 技术实现

### 分享流程
1. 用户在结果页点击"分享"按钮
2. App 调用 `handleShare()` 函数
3. 导入分享工具模块
4. 生成格式化文本
5. 调用系统分享 API
6. 用户选择分享目标
7. 完成分享或取消

### 错误处理
```typescript
try {
  await Share.open({...});
} catch (error) {
  if (error.message !== 'User did not share') {
    console.error('分享失败:', error);
  }
  // 用户取消不视为错误
}
```

### 平台适配
- **iOS**: 自动配置 LSApplicationQueriesSchemes
- **Android**: 自动配置 Queries
- **回退机制**: 目标应用未安装时自动回退到系统分享

// 在结果页面调用

  { year: '甲子', month: '乙丑', day: '丙寅', hour: '丁卯' },
