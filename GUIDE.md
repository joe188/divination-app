# Gitee Go 重新构建详细操作指南

## 步骤 1: 登录 Gitee 并访问仓库

1. 打开浏览器，访问 https://gitee.com
2. 登录您的 Gitee 账号
3. 访问仓库页面: https://gitee.com/joelinfo/divination-apk

## 步骤 2: 进入 Gitee Go 页面

1. 在仓库页面顶部找到 "Gitee Go" 标签并点击
2. 或者点击 "服务" → "Gitee Go"

## 步骤 3: 找到构建历史

1. 在 Gitee Go 页面找到 "构建历史" 或 "Build History"
2. 点击 "立即构建" 或 "Run Pipeline"

## 步骤 4: 重新构建

1. 点击 "立即构建" 按钮
2. 等待构建完成
3. 查看构建日志确认成功

## 步骤 5: 下载 APK

1. 构建成功后，在构建历史中找到产物下载
2. 点击 APK 文件下载

---
**提示**: 如果构建失败，请检查:
- Java 环境配置
- Gradle 配置
- 网络连接
